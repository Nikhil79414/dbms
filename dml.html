Data Manipulation Language (DML)


Data Manipulation Language (DML) is a database language, which enables user to manipulate data in the database. DML is a language used to insert, delete and modify database information. DML command is not auto-commited .

How to insert data in a table?

The insert command is used to insert new rows and records into a table. Values can be inserted for all columns or for the selected columns.

Syntax- INSERT INTO table_name(COLNAME, COLNAME....) VALUES (values,values......);

What is Like operator ?

The LIKE operator is used when you want to retrieve all rows in a table whose column values match a specified pattern. We use wild cards character "%" represent more than one character and "_" which represent single character.

Syntax-SELECT column_list FROM table_name WHERE like '';

How to work with null?

It is not possible to test for NULL values with comparison operators, such as =, <, or >. We will have to use the IS NULL and IS NOT NULL operators instead.

Syntax-SELECT column_names FROM table_name WHERE column_name IS NULL;

What is Distinct?

MySQL DISTINCT clause is used to remove duplicate records from the table and fetch only the unique records. The DISTINCT clause is only used with the SELECT statement.

Syntax-SELECT DISTINCT expressions FROM tables [WHERE conditions];

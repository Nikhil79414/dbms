View in MySql


How to create a view?

We can create a new view by using the CREATE VIEW and SELECT statement. SELECT statements are used to take data from the source table to make a VIEW.

Syntax-CREATE [OR REPLACE] VIEW view_name AS SELECT columns FROM tables [WHERE conditions];

How to delete a view?

We can drop the existing VIEW by using the DROP VIEW statement.

Syntax-DROP VIEW [IF EXISTS] view_name;

How to insert a value in a view?

We can insert the existing VIEW by using the insert statement.

Syntax- INSERT INTO VIEW view_name (values, values...);

How to create a read only view ?

We can create a read only view by using group function. In read only view you can't insert, delete and update values in view.

Syntax- CREATE VIEW view_name as SELECT group_function(COLNAME) FROM TABLE_NAME;

Example:



How to update a view ?

We can create a read only view by using update keyword.

Syntax- UPDATE VIEW set colname='value' where [condition];
